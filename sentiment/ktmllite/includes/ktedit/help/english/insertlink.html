Inserting Links



Links can be applied either to text selections or to pictures. You can link your page to other site pages, to external sites, to downloadable files or to an anchor within the same page. To insert a link, you have to select the desired text/image and use the Insert Link button from the toolbar.

You should first write a text in the Editable Area (even if this text represent a valid URL address) and then select it, press the Insert Link button and define the correct address (having the 'http://'antet) into the Link Inspector panel. This panel is automatically displayed under the Editable Region when you click on the "Insert Link" button. The Link Inspector is also displayed when you click inside a currently defined link.

Creating Anchor Links
You can also use the "Insert Link" feature to create an anchor link. This type of link redirects the visitor to another section of the same web page. To do this, you must first create the anchor to which the link will point:

  • select the text;
  • click on the Insert Link button;
  • type a name for the anchor in the Name field from the Property Inspector.

Next, you must create the actual link:

  • select the text you wish to use for the link;
  • click on the Insert Link button;
  • type "#anchor" in the Link field from the Property Inspector, where “anchor” must be replaced by the actual name you have given to your anchor.

Remember to always use the symbol #, followed by the name of the anchor, when you create anchor links.

Note: You cannot enter values in both the Name and the Href fields. The Name field is only used to create anchors, while the Href field is used to create the actual links. If you have already created a link and you want to edit the Name field, a dialog window will be displayed with the following message: "Do you want to transform this link into an anchor? (the old URL will be deleted)".
