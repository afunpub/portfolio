Undo & Redo

Undo
Undo allows you to go back to the stage previous to the last modification.
This feature is accessible either through the Undo button or by pressing the shortcut keys: "Ctrl+Z". KTML3 allows you to undo the last 30 changes.

Note: On Mozilla 1.4 browser, the Undo button does not work; you can use the shortcut key instead.

You can undo all the changes made on a page using KTML3:

  • Texts: text insertion, text deletion, text formatting, style formatting, font color changes, background color changes;
  • Images: image insertion, image resize, image properties modifications;
  • Tables: table insertions, table properties modifications, row insertions, column insertions, row and column properties changes, table deletion;
  • Links: link insertions, links properties changes;
  • Spellcheck: users can undo last word changes done by the spellchecker;
  • Clean HTML Content: users will be able to undo the changes operated by the Clean HTMLContent command.

Redo
Redo is the exact opposite of Undo. It can be applied only if the Undo command has been previously executed. It allows you to execute again the same action that you have previously canceled.
This feature is accessible in the Microsoft Internet Explorer browser either through the Redo button or by pressing the shortcut keys: “Ctrl+Y”.
You will be able to redo all the operations performed on a page using KTML3.

Note: In Mozilla, the Undo button does not work, only the shortcut keys do.
