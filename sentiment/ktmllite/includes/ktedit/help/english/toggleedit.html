Toggle Edit Mode – Show Code/Design

Because KTML3 is actually an WYSIWYG text editor for HTML documents, it was important to implement a function that will allow expert users to directly edit the document in HTML format. Therefore, the KTML3 editor has two editing modes: a text editor displaying the final result, and an HTML editor. To switch between these two modes, click on "Show Code/Design" button from the KTML3 toolbar.
