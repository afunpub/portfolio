Uploading Files

To create a link to a file, you should first upload the desired file to the remote server.

Write the file's local path or use the Browse button to navigate through local folders and select the file. You can upload only the supported file types: bmp, gif, jpeg, jpg, png, pdf, doc, rtf, txt, mp3, mpg, mpeg, avi, wav, mov, qt, rm.

Finally, click the Upload button to confirm your selection.
